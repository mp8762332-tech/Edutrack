## Step 2: School Admin Registration

**For School Administrators:**
1. Receive the 6-digit code from your HM
2. Click "School Onboarding" on the landing page
3. Enter the 6-digit code
4. Create admin account with email and password
5. You now have full access to the admin dashboard

---

## Step 3: Add Teachers & Students

### Add Teachers:
1. Go to Admin Dashboard → Teachers
2. Click "Add Teacher"
3. Enter teacher details (name, employee ID, subjects)
4. Teacher receives login credentials via email

### Add Students:
1. Go to Admin Dashboard → Students
2. Click "Add Student" or "Bulk Import CSV"
3. For CSV import, use this format:
   ```
   Name,Admission#,Class,Gender,DateOfBirth
   John Doe,ADM001,S3 East,Male,2008-01-15
   Jane Smith,ADM002,S3 East,Female,2008-03-22
   ```

---

## Step 4: Setup Classes & Subjects

### Create Classes:
1. Admin Dashboard → Classes
2. Click "Add Class"
3. Enter class name (e.g., "S3 East"), level, stream
4. Assign class teacher

### Add Subjects:
1. Admin Dashboard → Subjects
2. Click "Add Subject"
3. Enter subject name, code, level
4. Mark as Core or Elective

---

## Step 5: Configure Mid-Term Exam

### Create Exam Type:
1. Admin Dashboard → Exams
2. Click "Add Exam"
3. Enter:
   - Name: "Mid-Term Exam 2026"
   - Type: "Mid-Term" (optional)
   - Start Date: Your exam start date
   - End Date: Your exam end date
4. Save

### Important: End-of-Term Exam
- End-of-term exams are ALWAYS mandatory
- Cannot be deleted or made optional
- System enforces this automatically

---

## Step 6: Teacher Marks Entry

### For Teachers:
1. Login with your teacher credentials
2. Go to "Marks Entry"
3. Select Class and Subject
4. Enter marks for each student:
   - Paper 1 Mark (0-100)
   - Paper 2 Mark (0-100)
   - System auto-calculates average: (Paper1 + Paper2) / 2
5. Grade is auto-assigned based on average:
   - A: 80-100
   - B: 60-79
   - C: 49-59
   - D: 20-39
   - E: 0-19
6. Add teacher comments (optional)
7. Click "Save Marks"

---

## Step 7: Generate Report Cards

### For School Admin:
1. Admin Dashboard → Report Cards
2. Select Class
3. Click "Generate Report Cards"
4. System generates A4-ready report cards for all students

### Report Card Features:
- ✅ Print-perfect A4 format
- ✅ All boxes preserved when printed
- ✅ Professional layout
- ✅ Student marks, grades, teacher comments
- ✅ School logo and details

---

## Step 8: Export & Share Report Cards

### PDF Export:
1. Select a student's report card
2. Click "Download PDF"
3. Opens print-ready PDF

### Print on A4 Paper:
1. Open PDF in your browser
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select printer
4. Print settings:
   - Paper size: A4
   - Orientation: Portrait
   - Margins: Default
5. Boxes will remain untouched ✓

### Share via WhatsApp:
1. Click "Share via WhatsApp"
2. Pre-filled message with student name and marks
3. Choose recipient from your contacts
4. Send

### Email Report Cards:
1. Click "Send via Email"
2. Enter parent email
3. System sends report card as PDF attachment

---

## Step 9: Attendance Tracking

### Record Daily Attendance:
1. Teacher Dashboard → Attendance
2. Select Class and Date
3. Mark each student:
   - Present (P)
   - Absent (A)
   - Late (L)
4. Save

### View Attendance Reports:
1. Admin Dashboard → Attendance Reports
2. Filter by Class, Date Range
3. Export as CSV for records

---

## Step 10: Timetable Management

### Create Timetable:
1. Admin Dashboard → Timetables
2. Click "Add Timetable"
3. Select Type:
   - Academic (regular classes)
   - Mid-Term Exam (exam schedule)
   - End-of-Term Exam (final exam schedule)
4. Enter:
   - Class
   - Subject
   - Day of week
   - Start time (HH:MM)
   - End time (HH:MM)
5. Save

### View Timetable:
- Teachers see their assigned timetables
- Students see their class timetable
- Can print or export as PDF

## 🆘 Troubleshooting

### Can't Login?
1. Check email address is correct
2. Reset password using "Forgot Password"
3. Contact support if still issues

### Marks Not Saving?
1. Check internet connection
2. Refresh page (F5)
3. Try again
4. Contact support if persists

### Report Card Won't Print?
1. Use Chrome browser
2. Disable browser extensions
3. Clear browser cache
4. Try printing to PDF first

### Slow Performance?
1. Check internet speed (need 5Mbps+)
2. Close other applications
3. Clear browser cache
4. Try different browser
